#Crazier Alert

Snazz up your alerts with CRAAAAZZZ-IER ALERTS!

##Usage

```javascript
  // Load in with require
  define(function(require){
    var crazierAlert = require('gb-crazier-alert');
    crazierAlert("ayoooooo!");
    /* Output:
      WHOAAAAAA!!!!!!!!!!!!

      ayooooooo!
    */
  });
```

##Installation

```
  npm install gb-crazier-alert
```
